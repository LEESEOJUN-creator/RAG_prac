## 강의개요

This course is designed to provide students with an overview of Software Engineering, concentrating on practical aspects that are used for developing large and complex software systems. Key features are software process management, requirement engineering, system design, software testing, software code generation and code optimization.

소프트웨어공학 과목은 대규모의 복잡한 소프트웨어 시스템 개발에 사용되는 실용적인 측면에 집중하여 소프트웨어 공학의 개요를 학생들에게 제공하기 위해 설계되었습니다. 소프트웨어 프로세스 관리, 요구사항 공학, 시스템 설계, 소프트웨어 테스팅, 소프트웨어 코드 생성 및 코드 최적화가 주요 내용입니다.

## 강의목표

1) To understand fundamental principles in Software Engineering - 소프트웨어 공학의 기본 원리 이해
2) To know and practice techniques in Software Development Process - 소프트웨어 개발 프로세스 기법 습득 및 실습
3) To obtain abilities to resolve issues in Software Design Process - 소프트웨어 설계 프로세스의 문제 해결 능력 습득
4) To learn web based technologies and development process - 웹 기반 기술 및 개발 프로세스 학습
